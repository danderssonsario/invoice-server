# Resource Service
Självstående server-del som hanterar resurs i form av fakturor.

## Urval av tekniker.
* REST API
* Express js
* Puppeteer
* MongoDB / Mongoose
* jsonwebtoken
* Jest

## Script
### - npm run start
Kör node.
### - npm run dev
Kör node med nodemon.
### - npm run test:pdf
Kör tester från invoice.test.js
### - npm run test:invoice
Kör tester från pdf.test.js
